A Unity3D editor plugin for importing and playing general animation format files.
Author: Alexey Nikitin.
Copyright: © Catalyst Apps.
Version: 3.3.1

How to:
1)	Please remove all GAF asset files from the previouse plugin versions (this is neccessery for now sorry).
2)	Install it in Unity like any other plugin.
3)  Generate ".gaf" animation from your ".swf" ( For detailed info go to - http://gafmedia.com ).
4)	Copy a folder with your GAF animation files into a unity project directory.
5)	".gaf" file will be processed and animation asset file will be created.
6)	Create empty GameObject and add GAFMovieClip behavior to it.
7)	Drag’n’drop asset file onto GAFMovieClip interface field.
8)	Load animation.
